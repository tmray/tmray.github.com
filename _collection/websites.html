There Was a Fire - landing page [https://therewasafire.blogspot.com/](https://therewasafire.blogspot.com/) I created this page using Google's own Blogger platform. I decided to completely re-write the default code the was built inside Blogger, making my own customized landing page template on top of their code language. I decided to open up this process and created a repository called **Chainsaw**. It's a starting point for those that may also want to do this and still be able to use the Blogger CMS interface. Blogger re-write Example ------------------------ ``` <data:blog.pageTitle/> <data:blog.pageTitle/> ``` All the code examples can be viewed here.
